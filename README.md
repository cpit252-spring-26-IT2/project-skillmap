# SkillMap

## Description

SkillMap is a career roadmap web application that helps users create and manage their own personalized learning roadmaps. Users can manually build a roadmap based on their target role, add skills, certifications, weekly tasks, and supporting resources, then track their progress using checklists.

The project also includes verified predefined roadmap templates for common software engineering paths such as Frontend, Backend, and Mobile development. These templates can be used as a starting point and customized by the user.

## Features

- Custom roadmap builder.
- Add and manage skills.
- Add and manage certifications.
- Add weekly learning tasks.
- Add supporting resources and file references.
- Checklist-based progress tracking.
- Save roadmap history.
- Verified predefined roadmap templates.
